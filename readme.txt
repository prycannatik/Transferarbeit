Hinweise zur Benutzung des Programms
- Wählen Sie nach dem Start des Programms eine Textdatei zur Komprimierung aus.
- Das Programm erkennt automatisch, ob die Datei komprimiert oder dekomprimiert werden soll, und die entsprechende Schaltfläche wird aktiviert.
- Nach der Komprimierung wird der Ordner mit der Ausgabedatei automatisch geöffnet und die Kompressionsrate angezeigt.
- Der gleiche Vorgang wird beim Dekomprimieren durchgeführt.
